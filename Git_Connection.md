## Movimentação de arquivos  
---   

## Status  
	git status  
## Logs  
	git log  
## Detalhes  
	git diff  
 ### Criar Arquivo  
	touch <name.type>   
## Criar/Remover Pasta(Diretório)  
	mkdir <name>  
		rm <name> |ou| rm -f <name>  
## Salvar o Arquivo no Git  
	git add <name.type>  
## Snapshot(package)  
	git commit -m "Digite a mensagem"
